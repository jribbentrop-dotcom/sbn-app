Guitar music is notated in two visual shorthand systems: **tab** and **chord diagrams**. Both show the same information — which string, which fret — but from different perspectives.

<sbn-widget slug="tab-diagram" />

**The fretboard view** shows the neck horizontally, exactly as you see it while playing. Six strings run left to right, frets divide the neck into columns, dots mark where to press.

**Tab** removes the fret grid and keeps only the strings. Each horizontal line is a string, each number is a fret position. Tab reads left to right like text — it shows a melody or a chord sequence in time order.

**Chord diagrams** rotate the view 90°. Strings now run vertically top to bottom, frets run horizontally. This orientation matches what you see when you look down at your fretting hand. The thick top line is the nut; dots show finger placement; × marks muted strings, ○ marks open strings.

All three representations show the same chord — the Am shown here — just oriented differently. Recognising all three and switching between them fluently is an early but important skill.
